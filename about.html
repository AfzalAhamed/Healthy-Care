About Health Care

This is developed to manage customer's health management within Kurunegala district. In Colombo district we can see like these types of projects but other district we cannot find these types of projects. Therefore, we though to bring this type of system to Kurunegala district people. If this product is successful, we have an idea to launch on another district.

Functions allowed to be performed by the Customer

• Customer can manage account.
• Customer can order medicine.
• Customer can appoint doctors.
• Customer can use emergency services.

Functions allowed to be performed by Pharmacy

• Pharmacy can receive order.
• Pharmacy can update order.

Functions allowed to be performed by Delivery

• Delivery agent can receive order
• Delivery agent update order

Functions allowed to be performed by Channel Center

• Can receive appointments.
• Can Update appointments.
